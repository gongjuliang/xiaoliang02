🔐 小良内网穿透密码重置方法

用于本地调试和数据恢复的密码管理工具

⚠️ 重要提示

此工具仅用于本地合法调试和数据恢复目的。

请确保您有权操作相关数据库，遵守法律法规和项目安全规范。

📋 SQLite 数据库操作指南

1 定位数据库文件

NATT 项目的 SQLite 数据库文件通常位于以下位置：

组件 数据库路径
nattserver xiaoliang02_server/data/nattserver.db
nattuser xiaoliang02_user/data/nattuser.db

💡 如果找不到数据库文件，请先启动一次 NATT 服务，系统会自动创建数据库。

2 打开数据库

您可以使用以下工具之一打开 SQLite 数据库：

  • DB Browser for SQLite（推荐）：图形化工具，易于使用
  • sqlite3 命令行工具：在终端中运行 sqlite3 数据库文件路径
  • VS Code 插件：SQLite Viewer 或 SQLite Editor
3 查询用户信息

执行以下 SQL 语句查看所有用户：

-- 查询所有用户
SELECT id, username, password_hash, role, created_at FROM users;
4 更新密码哈希

使用下方生成的密码哈希值，执行以下 SQL 更新语句：

-- 更新指定用户的密码（将 '用户名' 替换为真实目标用户名，将 '新生成的哈希值' 替换为实际哈希）
UPDATE users SET password_hash = '新生成的哈希值' WHERE username = '用户名';

✅ 更新完成后，请使用新密码登录系统验证是否成功。
