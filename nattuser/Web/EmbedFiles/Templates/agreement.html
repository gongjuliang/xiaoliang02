第二条 服务内容与定义

2.1 软件介绍

本软件是一套基于 Go 语言的 C/S 架构 TCP 内网穿透工具,包括:

  • 小良内网穿透-服务端(nattserver):部署在公网服务器,负责端口监听、客户端授权、隧道管理
  • 小良内网穿透-客户端(nattuser):部署在内网设备,负责连接服务端并转发流量

2.2 核心功能

  1. TCP 端口穿透服务
  2. Web 控制台管理
  3. MCP(Model Context Protocol)接口支持
  4. 隧道状态监控和流量统计
  5. 审计日志记录

2.3 技术限制

当前版本仅支持:

  • TCP 协议(不支持 UDP、HTTP 域名转发)
  • 点对点隧道(不支持 P2P 打洞)
  • 明文传输(不包含 TLS 加密)
